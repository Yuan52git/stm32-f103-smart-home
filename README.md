# stm32-f103-smart-home
STM32F103C8T6、FreeRTOS、SU-03T 离线语⾳、PAJ7620 ⼿势传感器、DHT11、L298N、PWM、OLED  驱动开发：⼿写 DHT11 单总线驱动，通过调整采样延时解决校验和错误。基于状态机解析 SU-03T 串⼝协议（A5 XX 5A）， 采⽤中断 + 信号量同步任务。移植 PAJ7620 ⼿势传感器 I2C 驱动，完成 150+ 寄存器初始化。 系统设计：基于 FreeRTOS 划分为传感器采集、语⾳处理、⼿势识别、电机控制、显⽰刷新等 7 个任务，使⽤队列和信号量 实现任务间通信。设计⾃动温控状态机（28℃开、25℃关滞回控制），避免频繁启停。
